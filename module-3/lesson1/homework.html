Intro to Bootstrap

Fact 1: Responsive Design
Bootstrap’s grid system adjusts your layout for all screen sizes — phones, tablets, desktops.
Fact 2: Pre-built Components
Buttons, navbars, modals, and more ready for you to use with simple classes.
Fact 3: Utility Classes
Quickly style elements with margin, padding, colors, backgrounds, and more helpers.
Fact 4: Easy to Learn
Simple class names and great docs help you build professional sites fast.
